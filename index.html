Return Handling
Define how a customer return affects the salesperson's target credit and incentive for the period.
Target Credit on Return
Deduct from current period Deduct from original period No deduction
Recommended: current period adjustment — consistent with how other periods are handled and avoids reopening closed periods.
Incentive on Return
Clawback (reverse immediately) Adjust next period Lock after return window
Clawback is cleanest — commission is reversed as soon as the return is processed.
Return Window (Days)
Returns after this window do not affect target credit or incentive — the sale is permanently locked.
Partial Returns
Deduct returned value only Deduct full bill
For partial returns, only the returned item value is reversed from the salesperson's credit.
